**Academic Success Analysis**

**Project Overview**

This project analyzes key behavioral, social, and academic factors that influence student performance. Using the Student Performance dataset, the goal is to identify the most important predictors of academic success and build statistical models to estimate final grades.

**Dataset**

Source: Student Performance Dataset (UCI-style dataset)
File: student-por.csv
Observations: Student-level academic records
Features include:
- Study time
- Absences
- Family support
- Parental education
- Social and behavioral factors

**Objectives**
- Explore relationships between student characteristics and academic performance
- Identify key drivers of student success
- Build predictive models to estimate final grades
- Interpret statistical relationships in an educational context

**Methods & Approach**
- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Correlation analysis

**Key Insights**

- Academic habits such as study time and absences are strong predictors of performance
- Social and family-related factors show varying levels of influence
- Behavioral variables provide meaningful explanatory power in predicting outcomes
- A combination of features performs better than single-variable predictions

**Results**

- Built regression models to estimate student performance
- Identified statistically significant predictors of academic success
- Evaluated model fit using standard diagnostic measures

**Tools & Technologies**

- R
- dplyr
- ggplot2
- base R stats
- Data visualization libraries
